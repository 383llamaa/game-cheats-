> HELP --RUST\_USAGE



\[ 1. CALIBRATION ]

Rust recoil depends heavily on your IN-GAME SENSITIVITY. 

\- If the gun still goes UP: Increase 'v' in the script (e.g., 5.8 -> 6.2).

\- If the gun pulls DOWN: Decrease 'v' (e.g., 5.8 -> 5.4).



\[ 2. WEAPON SPECIFICS ]

\- AK-47 (302): This script compensates for the 'Z' kick. If you find the 

&nbsp; horizontal pull is too strong, reduce the -1.4 value.

\- M249 (301): This is a "Laser" script. It pulls down hard and fast. 

&nbsp; Recommended for use with a 4x scope.



\[ 3. STEALTH ADVICE ]

Rust Console monitors for "Static Pull." The random.uniform(-0.4, 0.4) 

ensures your crosshair "shakes" slightly while firing, which mimics 

natural hand tremor.
